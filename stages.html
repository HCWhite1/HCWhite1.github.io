Projet 2 — 4 semaines

// Création from scratch

Site de gestion des stocks interne DNI

La DNI gérait ses stocks via un Google Sheets incomplet et non-exhaustif. J'ai développé entièrement un outil web dédié, en choisissant volontairement des technologies nouvelles pour moi — Node.js + Express — afin d'élargir mes compétences au-delà de PHP.

  • Tableau de bord : vue d'ensemble des stocks et commandes, indicateurs d'alerte en temps réel
  • Gestion des stocks : catégories avec images, quantités par site (Douains / Mairie de Vernon / PM), seuil minimum configurable
  • Alertes automatiques : passage sous le seuil → alerte visuelle + création automatique d'une commande
  • Suivi des commandes : 7 statuts (à passer, en cours, en retard, terminé, partiel, retour, erreur), pièces jointes (PDF/image)
  • Validation d'une commande → incrémentation automatique du stock concerné
  • Gestion des accès : 2 niveaux (Super Admin / Admin), mots de passe hachés
  • Documentation intégrée, accessible directement depuis l'interface

Stack

Node.js Express.js Bootstrap MySQL

Projet principal — Semaines 3 à 5

// Création from scratch

Site Annuaire interne

Les agents s'appuyaient sur des fichiers Excel ou des images pour retrouver leurs collègues — peu pratique et difficile à maintenir. J'ai conçu et développé entièrement un annuaire web centralisé, hébergé en local sur WampServer, couvrant cinq entités : SNA, Vernon, CIAS, OTC et CCAS.

  • 3 niveaux d'accès : lecteur (consultation seule) / modificateur (CRUD complet) / administrateur (gestion des comptes)
  • Recherche globale multi-colonnes : combinaison libre de mots-clés sur toutes les colonnes simultanément
  • Filtres avancés : menus déroulants par entité, service, pôle, direction
  • Basculement interne / externe pour les contacts hors-structure
  • Mode sombre persistant (stockage local)
  • Import CSV en masse
  • Champ localisation par personne (bureau, bâtiment, étage…)
  • Système de logs traçant toutes les actions (heure, action, utilisateur)
  • Mots de passe stockés sous forme hachée

Stack

HTML / CSS PHP JavaScript MySQL WampServer
